### Sobre mi   

Soy desarrollado web full stack graduado del bootcamp de Henry, soy tecnólogo en mecatrónica y actualmente curso 9no semestre de ingeniería electrónica.

Gracias a Henry pude adquirir conocimientos suficientes para el desarrollo de aplicaciones web full stack, me gusta mucho la parte del frontend, la experiencia de usuario y el diseño gráfico de las aplicaciones.

Disfruto mucho de programar, trabajar en equipo y trabajar para que un proyecto salga adelante, estoy dispuesto a siempre seguir aprendiendo y enfrentarme a cualquier reto que se presente.
